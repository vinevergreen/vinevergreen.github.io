uploading

the Evergreen tree comes pre-provisioned to upload its output to a GitHub Pages site. to do this, open a terminal, cd to the root of the Evergreen project, and run the command ./up.sh.

the script uses an OAuth token to communicate with the GitHub server so you don't need to enter your password when you update the site. however, the default password to your GitHub account is stored in the default-password file; you should change this at GitHub when you have the chance.

if you don't need to check the local output before you update the site, you can use the command ./pub.pl; ./up.sh to accomplish in one go.
